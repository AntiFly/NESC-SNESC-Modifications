=== BlueMSX core for RetroArch ===

This module will add support for Coleco Vision & MSX games to RetroArch
The BIOS files must be inside the game directory or Via Bios Module where applicable

Available executables and arguments:
- /bin/col <rom> <clover_args>
  runs "bluemsx" core
- /bin/msx <rom> <clover_args>
  runs "bluemsx" core
- /bin/bluemsx <rom> <clover_args>
  runs "bluemsx" core
