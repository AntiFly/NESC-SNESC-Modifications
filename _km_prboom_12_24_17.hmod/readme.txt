=== PrBoom core for RetroArch ===

This module will add support for Doom Games to RetroArch

The following image is required for this core to work properly:
prboom.wad  Copy it into pertaining Game Directory after Adding via Hakchi

Available executables and arguments:
- /bin/wad <rom> <clover_args>
  runs "prboom" core
- /bin/prboom <rom> <clover_args>
  runs "prboom" core
