=== SNES9x2002 core for RetroArch ===

This module will add support for Super Famicom / Super Nintendo games to RetroArch

This also allows Dual-Tri-Stackable Usage with SNES9x 9x/2002/2005/2010, also aptly titled "dual-core"!

Available executables and arguments:
- /bin/snes <rom> <clover_args>
  runs "snes9x2002" core
- /bin/snes02 <rom> <clover_args>
  runs "snes9x2002" core
