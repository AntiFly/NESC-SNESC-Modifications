=== Vice x64 core for RetroArch ===

This module will add support for Commodore 64 to RetroArch

Available executables and arguments:
- /bin/c64 <rom> <clover_args>
  runs "vice_x64" core
- /bin/vicex64 <rom> <clover_args>
  runs "vice_x64" core
