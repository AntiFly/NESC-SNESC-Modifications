=== PicoDrive core for RetroArch ===

This module will add support for Sega CD, Master System, Mega Drive / Genesis, Game Gear and Sega 32X games to RetroArch

Available executables and arguments:
- /bin/32x <rom> <clover_args>
  runs "picodrive" core
- /bin/gg <rom> <clover_args>
  runs "picodrive" core
- /bin/md <rom> <clover_args>
  runs "picodrive" core
- /bin/picodrive <rom> <clover_args>
  runs "picodrive" core
- /bin/scd <rom> <clover_args>
  runs "picodrive" core
- /bin/sms <rom> <clover_args>
  runs "picodrive" core
