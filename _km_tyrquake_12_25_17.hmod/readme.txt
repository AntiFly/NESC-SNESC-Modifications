=== TyrQuake core for RetroArch ===

This module will add support for Quake Games to RetroArch

Available executables and arguments:
- /bin/pak <rom> <clover_args>
  runs "tyrquake" core
- /bin/tyrquake <rom> <clover_args>
  runs "tyrquake" core
