=== MAME2003 - Standard Variant Core for RetroArch ===

Module adds support for MAME 2003

Bios must be inside game directory or via Bios Module where applicable

This Particular MAME Variant also Offers Very Slow KI/KI2 Support!

Available executables and arguments to run Core:
- /bin/mame2003 <rom> <clover_args>
- /bin/mame2003-standard <rom> <clover_args>
