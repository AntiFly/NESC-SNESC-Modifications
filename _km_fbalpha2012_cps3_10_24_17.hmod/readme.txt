=== Final Burn Alpha 2012 CPS3 core for RetroArch ===

This module will add support for CP System III arcade games to RetroArch
The BIOS files must be inside the game directory or Via Bios Module where applicable
7zip Support does not function with this Variant.  But, it does with FBA2012/2016

Available executables and arguments:
- /bin/cps3 <rom> <clover_args>
  runs "fb_alpha_cps3" core
