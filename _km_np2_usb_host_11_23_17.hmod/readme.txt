=== NekoP2 core for RetroArch ===

This module will add support for NEC PC-98 games to RetroArch.

Available executables and arguments:
- /bin/pc98 <rom> <clover_args>
  runs "nekop2" core
- /bin/np2kai <rom> <clover_args>
  runs "nekop2" core
