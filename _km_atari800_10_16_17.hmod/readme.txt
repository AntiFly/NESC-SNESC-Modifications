=== Atari800 core for RetroArch ===

This module will add support for Atari 8-bit computer systems and 5200 to RetroArch

Available executables and arguments:
- /bin/a52 <rom> <clover_args>
  runs "atari800" core
- /bin/atari800 <rom> <clover_args>
  runs "atari800" core
