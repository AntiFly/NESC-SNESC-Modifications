=== ProSystem for RetroArch ===

This module will add support for Atari 7800 games to RetroArch
The following BIOS image is optional for this core to work properly:
7800 BIOS (U).rom

Available executables and arguments:
- /bin/a78 <rom> <clover_args>
  runs "prosystem" core
