=== TGB Dual core for RetroArch ===

This module will add support for Dual Link Game Boy and Game Boy Color games to RetroArch

Available executables and arguments:
- /bin/gb <rom> <clover_args>
  runs "tgbdual" core
- /bin/gbc <rom> <clover_args>
  runs "tbgdual" core
- /bin/tgbdual <rom> <clover_args>
  runs "tbgdual" core
