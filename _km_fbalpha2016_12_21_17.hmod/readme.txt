=== Final Burn Alpha 2016 core for RetroArch ===

This module will add support for Final Burn Alpha arcade games to RetroArch
The BIOS files must be inside the game directory or Via Bios Module where applicable
7zip works with this Core

Available executables and arguments:
- /bin/fba <rom> <clover_args>
  runs "fbalpha" core
- /bin/fba2016 <rom> <clover_args>
  runs "fbalpha" core
